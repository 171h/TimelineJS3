TimelineJS is Knight Lab's most popular product, but its rapid and widespread adoption have made it hard to change some things about it. TimelineJS 3 is a full rewrite designed to make this popular tool more flexible and easier to enhance.

This site offers a preview of the new TimelineJS and an opportunity for more technical users to help us put the code through its paces and find obscure bugs. If you're a current user who is curious, you can see how your current timeline would look. If you're brand new, you should probably stick with the original TimelineJS. If you want to help us out, make a few timelines and let us know if anything goes wrong.

What's changed?

  • Flexible row layout: no longer limited to 3/6 rows.
  • Encapsulation: More than one timeline per page.
  • Support for very large time scales: dinosaurs! the Big Bang!
  • New media types: Dropbox images; DocumentCloud
  • Dropped support for Pagepeeker (screenshots of websites)

What's coming?

  • Spreadsheet-based data format
  • Support for "tags"
  • Support for "eras"
