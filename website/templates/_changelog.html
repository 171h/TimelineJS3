Help us out!

TimelineJS 3 is a public alpha, which means there are still some rough patches that need to be smoothed over. We need your help in discovering the issues that need to be addressed. Make a few timelines, and let us know if anything goes wrong.

What's changed?

  • Flexible row layout: no longer limited to 3/6 rows.
  • Encapsulation: More than one timeline per page.
  • Support for very large time scales: dinosaurs! the Big Bang!
  • New media types: Dropbox images; DocumentCloud
  • Dropped support for Pagepeeker (screenshots of websites)

What's coming?

  • Spreadsheet-based data format
  • Support for "tags"
  • Support for "eras"
