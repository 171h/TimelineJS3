Test-drive TimelineJS 3

If you have existing timelines, you can enter the Google spreadsheet URL here to see how they look and function in TimelineJS 3. This is the same thing you'd enter in step 3 of the current TimelineJS creator.

While TimelineJS 3 is still in "alpha", we aren't offering an iframe embed tool. If you're comfortable with a more hands-on approach, see this guide. If you really want to go deep, see the GitHub repository.

Note on backwards compatibility

TimelineJS 3 is not backwards-compatible with all media. Services that have updated their URL structures, such as Google Maps, will only appear when the timeline uses the most recent structure.
