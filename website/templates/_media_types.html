Media Types

People have always loved how easy TimelineJS makes it to add rich media to the slides that tell your story. The new version of Timeline continues the tradition and adds a few new media types.

Image
If your URL ends in jpg, gif, png or jpeg, it will be used as the source for an image tag.
YouTube
Use the URL for the YouTube page of the video. Timeline honors start-at time parameters.
Vimeo
Use the URL for the Vimeo player page.
DailyMotion
Use the URL for the DailyMotion player page.
Vine
Use the URL for the Vine player page.
SoundCloud
Use the URL for the SoundCloud player page.
Twitter
Use the URL provided in the Twitter embed code.
Google Maps
Use the URL for the Google Maps page.
Google Plus
Use the URL for the Google Plus photo's page.
Instagram
Use the URL for the Instagram photo's page.
Flickr
Use the URL for the Flickr photo's page.
